라이브러리란? = 자주 사용하는 로직을 재사용할 수 있도록 고안된 소프트웨어를 라이브러리라고 한다.
이유: jQuery는 DOM(문서 객체 모델)을 내부에 감추고 보다 쉽게 웹페이지를 조작할 수 있도록 돕는 도구이다. - 자주사용하는 패턴, 작업들을 효율적으로 할 수 있다.
단점: 하지만 이것만 알게되면 라이브러리에서 제공하는 기능에 갖칠 수 있다.


jQuery를 사용하기 위해서는 jQuery를 담고있는 파일이 있어야한다. CDN = content delivery network = 제이커리가 제공하는 서버를 네트워크로 사용하는 것


jquery의 문법은 단순하고 강력하다.
jquery는 $(함수)로 시작한다. jquery 함수의 인자로는 일반적으로 css 선택자가 사용된다.
ex] $('li').css('color', 'red');
.뒤의 것은 객체라고 하는데 css라는 메소드를 호출하면 jqeury 객체를 만든 jqeury함수의 인자에 해당되는 엘리먼트(=li) 전체에 대해서 css가 color을 red로 바꾸는 역할을 한다. 즉
  • 로 만든다는 것이다.
